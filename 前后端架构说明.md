# 前后端分离架构 - 启动指南

## 项目结构

```
claude_content_agents/
├── backend/              # FastAPI 后端
│   ├── main.py          # API 主入口
│   └── requirements.txt # Python 依赖
├── frontend/            # React 前端
│   ├── src/
│   │   ├── App.jsx     # 主组件
│   │   └── main.jsx    # 入口文件
│   ├── package.json    # Node 依赖
│   └── vite.config.js  # Vite 配置
└── src/                # 原有 Agent 逻辑
```

## 快速启动

### 1. 启动后端

```bash
# 安装依赖
cd backend
pip install -r requirements.txt

# 启动服务 (默认端口 8000)
python backend/main.py
```

访问 http://localhost:8000/docs 查看 API 文档

### 2. 启动前端

```bash
# 安装依赖
cd frontend
npm install

# 启动开发服务器 (默认端口 3000)
npm run dev
```

访问 http://localhost:3000 查看界面

## 功能特性

### REST API
- `POST /api/generate` - 提交文案生成任务
- `GET /api/task/{task_id}` - 查询任务状态

### WebSocket
- `WS /ws/task/{task_id}` - 实时接收任务进度

### 前端界面
- 响应式设计
- 实时进度展示
- Agent 步骤状态可视化
- 结果展示

## 技术栈

**后端**: FastAPI + Uvicorn + WebSocket
**前端**: React 18 + Vite + Tailwind CSS
**通信**: REST API + WebSocket 实时推送
